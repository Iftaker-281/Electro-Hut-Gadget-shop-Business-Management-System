# Electro Hut - Gadget Inventory & Order Management System

## Project Overview

**Electro Hut** is a Java-based console application built using **Object-Oriented Programming (OOP)** and **File Handling**.

It is designed to manage:
- Products
- Orders
- Payments
- Resellers

 This system stores all data in `.txt` files using a custom file handler.

## Features
-  Add New Products  
- View Products  
- Place & Manage Orders  
- Payment Handling  
- Reseller Management  
- File-based Persistent Storage  

## Technologies Used
- Java (OOP Concepts)
- File Handling (FileReader / FileWriter)
- Console-Based UI
